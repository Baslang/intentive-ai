# Intentive AI

## Overview

Intentive AI is a contextual recommendation intelligence system designed for Task B of the competition.

Unlike traditional recommendation systems, Intentive AI focuses on:

* deep intent understanding
* emotional reasoning
* contextual recommendation ranking
* semantic retrieval
* personalized reranking

The system reasons before recommending.

---

## Architecture

User Input
→ Intent Understanding
→ Memory Retrieval
→ Context Analysis
→ Embedding Search
→ Candidate Filtering
→ Scoring + Reranking
→ Recommendation Generation
→ Feedback Adaptation

---

## Core Features

### Intent Understanding

Analyzes:

* surface intent
* latent intent
* emotional state
* behavioral patterns
* novelty tolerance
* cognitive preference

### Contextual Ranking

Recommendations are ranked using:

* semantic similarity
* emotional alignment
* contextual relevance
* behavioral consistency
* novelty compatibility

### Cold-Start Handling

The system handles new users through:

* conversational understanding
* semantic reasoning
* embedding-based retrieval

### Recommendation Intelligence

The system uses:

* embeddings
* candidate filtering
* scoring logic
* reranking
* contextual reasoning
